# AnUnexpectedHelpEducationalMathsGame
Το εκπαιδευτικό παιχνίδι αποσκοπεί στην επίλυση ασκήσεων και προβλημάτων που βασίζονται στην ύλη των Μαθηματικών Δ' Δημοτικού. Το περιβάλλον του παιχνιδιού είναι φιλικό και κατάλληλο για τις ηλικίες των παιδιών καθώς περιέχει χαρακτήρες από κομικ που τους ενθαρρύνουν κατά την πορεία στο παιχνίδι.

Link εκπαιδευτικού παιχνιδιού: https://drive.google.com/file/d/1NZdDw8pP47Rkkrz4iyTH4XYsTHgetONv/view?usp=drive_link
